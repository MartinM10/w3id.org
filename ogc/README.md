# /ogc
This [W3ID](https://w3id.org) provides a persistent URI namespace for the [Open Geospatial Consortium](https://www.ogc.org).

The following sub-namespaces are defined:

* OGC
  * `/geopose`
* Hosted by OGC
  * `/hosted`
